## 🎨 採用的優化方案

### 選擇：方案 2 + 方案 3 混合方案

**理由**：
1. **保留所有資訊**：符合用戶要求，不刪除任何功能
2. **合理分層**：通過三行佈局分散資訊密度
3. **增加間距**：使用 `gap-2` 和 `gap-3` 提升可讀性
4. **突出重點**：訂閱狀態獨立顯示，更加醒目

### 具體實作

#### 桌面版佈局（三行結構）

**第一行：等級 + 活躍度 + 額度**
- 等級徽章（可點擊查看個人資料）
- 活躍度點數（帶顏色圖示）
- 上傳額度 / 查詢額度（灰色背景卡片）
- 使用 `gap-3` 間距

**第二行：會員資訊 + 訂閱狀態**
- 會員暱稱
- 分隔符號「|」（取代原本的「-」）
- 訂閱狀態標籤（VIP/免費體驗）
- 剩餘天數（帶顏色提示）
- 使用 `gap-2` 間距

**第三行：業務資訊**
- 業務類型 · 業務區域 · 審核狀態
- 使用 `gap-2` 間距

#### 手機版佈局（保持不變）
- 手機版已經是縱向排列，無需修改
- 訂閱狀態獨立一行顯示
- 視覺效果良好

## 🔧 技術實作

### 修改檔案
- `src/components/member/MemberNav.tsx`

### 程式碼變更

#### 主要結構變更
```tsx
// 優化前：橫向排列（<>...</>）
{member && !loading && (
  <>
    {/* 等級徽章 */}
    {/* 活躍度 */}
    {/* 額度 */}
    {/* 會員資訊（包含訂閱狀態）*/}
  </>
)}

// 優化後：三行分層（flex-col）
{member && !loading && (
  <div className="flex flex-col gap-2">
    {/* 第一行：等級 + 活躍度 + 額度 */}
    <div className="flex items-center gap-3">...</div>
    
    {/* 第二行：會員資訊 + 訂閱狀態 */}
    <div className="flex items-center gap-2">...</div>
    
    {/* 第三行：業務資訊 */}
    <div className="text-foreground-muted text-xs flex items-center gap-2">...</div>
  </div>
)}
```

#### 關鍵改進點

1. **外層容器**：
   - 從 `<>...</>` 改為 `<div className="flex flex-col gap-2">`
   - 使用 `flex-col` 實現縱向排列
   - 使用 `gap-2` 設定行間距（0.5rem = 8px）

2. **第一行容器**：
   - `<div className="flex items-center gap-3">`
   - 使用 `gap-3` 設定元素間距（0.75rem = 12px）
   - 保持原有的等級、活躍度、額度顯示

3. **第二行容器**：
   - `<div className="flex items-center gap-2">`
   - 分隔符號從「-」改為「|」，更簡潔
   - 移除多餘的 `<span>` 包裝，直接顯示訂閱狀態

4. **第三行容器**：
   - `<div className="text-foreground-muted text-xs flex items-center gap-2">`
   - 使用 `gap-2` 取代 `space-x-2`，更一致
   - 保持原有的業務資訊顯示

## 🎨 設計改進

### 間距系統
- **行間距**: `gap-2`（8px）- 適中，不會過於緊湊
- **元素間距**: `gap-2`（8px）或 `gap-3`（12px）- 根據內容密度調整
- **內邊距**: 保持原有的 `px-2 py-1` 或 `px-3 py-1`

### 分隔符號
- **優化前**: 使用「-」作為分隔符號
- **優化後**: 使用「|」作為分隔符號
- **優勢**: 更簡潔、更現代、視覺上更輕盈

### 顏色系統（保持不變）
- **VIP 標籤**: 金色漸層（#F59E0B → #D97706）+ 👑
- **免費試用標籤**: 灰色（#6B7280）
- **剩餘天數**:
  - 正常（> 7 天）: 白色
  - 警告（3-7 天）: 黃色（#EAB308）
  - 緊急（≤ 3 天）: 紅色（#EF4444）
